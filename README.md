# ⚛️🔥 Interactive Physics Learning Portal
### รายวิชา ฟิสิกส์ 5 (ว33205) | ฟิสิกส์ ม.6 บทที่ 16.4 กฎข้อที่หนึ่งของอุณหพลศาสตร์ (First Law of Thermodynamics)

โปรเจกต์สื่อการสอนแบบโต้ตอบ (Interactive Web Application) ที่ถูกออกแบบมาเพื่อช่วยให้นักเรียนระดับชั้นมัธยมศึกษาปีที่ 6 เข้าใจความสัมพันธ์ของการถ่ายโอนความร้อน ($Q$), การเปลี่ยนแปลงพลังงานภายในระบบ ($\Delta U$) และงานที่ทำโดยแก๊ส ($W$) ผ่านการจำลองภาพ (Simulation) การฝึกทำโจทย์แบบสุ่มตัวเลข และระบบสอบเก็บคะแนนแบบจับเวลา ตัวแอปพลิเคชันถูกรวมไว้ในไฟล์ HTML เพียงไฟล์เดียว (Single-File HTML) เพื่อความสะดวกในการใช้งานแบบออฟไลน์หรือนำไปโฮสต์บนเว็บเซิร์ฟเวอร์ทั่วไป

### 1. ✨ ฟีเจอร์หลัก (Key Features)
<details>
<summary>คลิกเพื่อดูรายละเอียด 📖 🧠 ⏱️</summary>
<br>

#### 1.1 📖 ระบบทบทวนบทเรียนแบบโต้ตอบ (Interactive Review)
*   **Simulator 1: พลังงานภายในระบบ ($\Delta U$)**
    *   แบบจำลอง Canvas 2D แสดงการเปลี่ยนแปลงพลังงานภายในระบบของแก๊สอุดมคติที่แปรผันตรงกับอุณหภูมิสัมบูรณ์ ($T$)
*   **Simulator 2: งานที่ทำโดยแก๊ส ($W$)**
    *   แบบจำลองกระบอกลูกสูบ แสดงการขยายตัวและหดตัวของแก๊ส เพื่อทำความเข้าใจทิศทางและเครื่องหมายบวก/ลบของงานเมื่อปริมาตรเปลี่ยนแปลง
*   **เครื่องคำนวณอุณหพลศาสตร์ ($Q = \Delta U + W$)**
    *   เครื่องมือช่วยคำนวณกฎข้อที่หนึ่งของอุณหพลศาสตร์ ช่วยวิเคราะห์และสรุปทิศทางการถ่ายโอนพลังงานพร้อมสีแจ้งเตือนสถานะชัดเจน

#### 1.2 🧠 ระบบตะลุยโจทย์ (Dynamic Practice Mode)
*   **ระบบสุ่มตัวแปรในโจทย์ (RNG):** ผู้เรียนได้ฝึกคิดเลขและคำนวณโจทย์ที่ไม่ซ้ำกัน อิงจากเลขที่ของนักเรียน ($R$)
*   **ขอบเขตเนื้อหา:** ครอบคลุมหัวข้อย่อย 16.4.1 (พลังงานภายใน), 16.4.2 (งานของแก๊ส) และ 16.4.3 (กฎข้อที่ 1 และเครื่องหมายอุณหพลศาสตร์)
*   **มาตรฐานการประเมิน:** อิงตามแนวทางแบบฝึกหัดและข้อสอบ สสวท.
*   **เฉลยคำตอบ:** มีระบบตรวจคำตอบทันที พร้อมแสดงเฉลยวิธีทำทีละขั้นตอน (Step-by-step Explanation) ด้วย LaTeX ที่อ่านง่าย

#### 1.3 ⏱️ ระบบสอบเก็บคะแนน (Timed Exam System)
*   **จำลองการสอบ:** ทดสอบ 5 ข้อ (10 คะแนน) พร้อมตัวจับเวลาถอยหลัง (Timer)
*   **ระบบป้องกันทุจริต:** ป้องกันการออกจากหน้าสอบหรือรีเฟรชหน้าจอ (Anti-cheat Guard)
*   **สรุปผลสอบ:** แสดงคะแนนและ Dashboard วิเคราะห์ความถูกต้องของแต่ละข้อ พร้อมวิธีทำอย่างละเอียด
*   **ระบบบันทึกความก้าวหน้า:** บันทึกผลสอบล่าสุดไว้ในเบราว์เซอร์ (Local Storage) อัตโนมัติ

</details>

### 2. 🧩 ข้อกำหนดการพัฒนาคำถาม (Question Spec)
<details>
<summary>คลิกเพื่อดูข้อกำหนดและเงื่อนไขการสุ่มโจทย์ 🛠️</summary>
<br>

*   **Dynamic Parameter Generation:** ค่าของตัวแปรในโจทย์จะถูกสร้างขึ้นแบบไดนามิก (Dynamically) จากเลขที่ $R$ ของนักเรียน เพื่อป้องกันการท่องจำคำตอบ (Static Hardcoded)
*   **Safety Constraints:** การสุ่มพารามิเตอร์ที่ผูกกับ $R$ มีระบบคัดกรองไม่ให้คำตอบติดลบ หรือเกิดค่าที่ไม่สมเหตุสมผลทางฟิสิกส์

</details>

### 3. 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
<details>
<summary>คลิกเพื่อดูชุดพัฒนาโปรแกรมและไลบรารี 💻</summary>
<br>

*   **HTML5 / CSS3 / JavaScript (ES6+):** เขียนด้วย Vanilla JS ล้วนๆ โดยไม่พึ่งพิง Framework ใหญ่ๆ ทำให้โหลดหน้าเว็บได้รวดเร็วและน้ำหนักเบา
*   **Tailwind CSS (via CDN):** ใช้จัดการ UI/UX สีสัน เลย์เอาต์ และให้แอปพลิเคชันรองรับการใช้งานทุกขนาดหน้าจอ (Responsive Design) ทั้ง PC, แท็บเล็ต และสมาร์ทโฟน
*   **MathJax:** ไลบรารีสำหรับประมวลผลและแสดงผลสมการคณิตศาสตร์/สูตรฟิสิกส์ในรูปแบบ LaTeX ได้อย่างสวยงามบนเว็บเบราว์เซอร์
*   **FontAwesome:** ชุดไอคอนสัญลักษณ์ต่างๆ ที่ช่วยให้สื่อสารรูปแบบฟีเจอร์ได้น่าใช้งานยิ่งขึ้น
*   **Google Fonts:** เลือกใช้ฟอนต์ `'Prompt'` และ `'Sarabun'` เพื่อผสมผสานความทันสมัยและความน่าเชื่อถือทางวิชาการ

</details>
